Structured Data: Mapping an address to a struct allows you to store multiple related variables (like balance and status) under a single key.Instant Validation: Mappings allow you to check a user's isActive status in $O(1)$ time without searching through a list of users.Default State Check: In Solidity, if a user hasn't been created yet, their isActive bool in the mapping will be false by default.Storage Pointer: When retrieving a struct from a mapping to modify it, using the storage keyword creates a reference that lets you update the blockchain's state directly.
