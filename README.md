# 🚀 Blockchain Group Project

## 📘 Project Overview

This repository contains our group’s work for the **Learn Solidity** course from **Alchemy University**.

Throughout this course, we explored the fundamentals of blockchain development, smart contract programming, and Ethereum-based application logic using **Solidity**.

Our goal was to build practical smart contract examples, document key concepts, and organize all learning materials in a structured repository.

---

## 🎯 Group Objectives

- Complete the full Learn Solidity course
- Practice Solidity by building multiple smart contracts
- Understand blockchain fundamentals and contract architecture
- Organize all code, notes, and screenshots in one shared repository
- Collaborate as a team using Git and GitHub
