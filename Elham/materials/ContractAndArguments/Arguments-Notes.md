# Solidity Materials: Constructor Arguments

## Key Concepts
* **Constructor**: A special function that runs only once during contract deployment.
* **Arguments**: Parameters passed during deployment to initialize the contract state.
* **Variable Shadowing**: Occurs when a local variable (parameter) has the same name as a storage variable. 
* **Naming Convention**: Using an underscore (e.g., `_x`) for parameters is a common practice to avoid shadowing.
