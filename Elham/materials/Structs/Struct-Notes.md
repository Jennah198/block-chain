# Solidity Structs
* **Custom Types**: Structs allow you to create your own data structures by grouping different variables together.
* **Initialization**: You can create a struct instance by passing values in order `Hero(Up, 100, true)` or by naming them `Hero({health: 100, ...})`.
* **Access**: Use the dot operator (`.`) to access specific properties within a struct.
* **Storage**: Structs can be stored in state variables, making them persistent on the blockchain.
