# Solidity Materials: Enum Types

## Key Concepts
* **Readability**: Replaces "magic numbers" (0, 1, 2) with human-readable names.
* **Safety**: Restricts a variable to a predefined set of constants.
* **Under the Hood**: Internally, Solidity treats enums as integers (starting from 0).
* **Interoperability**: Enums can be shared across different contracts to maintain consistency.
